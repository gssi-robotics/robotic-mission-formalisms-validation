Mission concepts mapping to SM
Mission concept Mapping to SM
Skill Simple State
Task Composite State
Data Data handled inside states and global variables configuration
Communication Not explicitly modeled. It relies on the implementation of states behavior
Event Transitions may be associated with events, but they only activate if the related state is currently active
Error A transition may represent an error, but it only activates if the related state is currently active
Pre/post-conditions Modeled through transition guards; a specialization named "Protocol Transition" supports pre- and post-conditions
